# 阻容网当前最新基线记录

- 当前基线来源：用户上传的 `zr.rar`
- 处理时间：2026-06-12（北京时间）
- 基线说明：以本次发布包为最新网站基线，旧入口、原型残留、模拟登录和过期文案不再作为开发依据。
- 已同步规则：
  - 所有数量输入只允许不以 `0` 开头的正整数。
  - 禁止 `0`、`01`、负数、小数、中文和非数字内容。
  - 列表、卡片、记录中涉及数量、库存、需求量、单价时，数字与单位分开展示。
  - 工作台访问控制交给后端登录会话和 `members.role`，前端不使用本地模拟登录。
  - 数据库统一命名为 `zr_platform`，完整建表脚本见 `zr-database-schema.sql`。
  - 全站开发逻辑、页面映射和发布说明见 `zr-development-doc.html`。
